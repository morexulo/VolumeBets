# Tipster Analytics Dashboard

Proyecto en Streamlit para analizar resultados de apuestas:
- ROI por tipo de apuesta
- Equity curve
- Win-rate por deporte
- Ranking de mercados rentables

Estructura:
- app/main.py → Dashboard principal
- data/ → CSVs
- assets/ → imágenes, logos, etc.
- utils/ → funciones auxiliares
